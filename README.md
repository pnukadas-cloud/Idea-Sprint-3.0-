# Idea-Sprint-3.0-
This project is an AI-based event scheduling system that dynamically creates and updates event plans based on constraints.

## 🚀 Features
- Speaker scheduling
- Budget constraint handling
- Room capacity optimization
- Dynamic re-planning
- AI-based decision explanation

## 🛠️ Tech Stack
- Python
- flask
